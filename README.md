# AI HIRE 🚀

Welcome! If you have zero knowledge of AI, you are in the perfect place. By the time you finish exploring this repository, you will clearly understand what Machine Learning, Deep Learning, and Generative AI are, how we train them, and what tools we use to build them.

Please explore the folders in this exact order:

## 📁 1. [Machine Learning (ML)](ML_Notes_Training/)
**What is it?** Teaching computers to find patterns in data (like spreadsheets) to make predictions, rather than giving them explicit rules.
*   *Go into this folder to learn about Scikit-Learn and basic model training.*

## 📁 2. [Deep Learning (DL)](DeepLearning_Notes_Training/)
**What is it?** A powerful sub-field of ML inspired by the human brain. We use "Artificial Neural Networks" to solve much harder problems like image recognition or processing complex text.
*   *Go into this folder to learn about TensorFlow, Keras, and Neural Networks.*

## 📁 3. [Generative AI (GenAI)](GenAI_Notes_Training/)
**What is it?** The cutting-edge of AI. Instead of just predicting a number or category, these models (like ChatGPT) *generate* brand new text, code, or images.
*   *Go into this folder to learn about LangChain, Prompts, and building autonomous AI Agents.*
